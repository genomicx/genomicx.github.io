About Genomicx

Open-source browser bioinformatics, built by a researcher who needed them to exist.

The Project

Genomicx is a growing collection of bioinformatics web applications built to run entirely in the browser using WebAssembly. Each tool compiles established command-line bioinformatics software (like minimap2, LASTZ, and Mash) into WebAssembly modules, letting researchers run real analyses without installing software, configuring servers, or uploading sensitive data to third-party services.

The goal is simple: make everyday genomics tasks as easy as opening a web page. Every tool is free, open source, and designed to work on any modern browser.

Nabil-Fareed Alikhan

Nabil-Fareed Alikhan is a bioinformatics researcher and software developer at the Centre for Genomic Pathogen Surveillance, University of Oxford. His work focuses on microbial genomics, developing widely-used open-source tools for prokaryote genome analysis and visualization.

He is the co-creator of BRIG (BLAST Ring Image Generator), published in BMC Genomics, and co-hosts the MicroBinfie Podcast exploring microbial bioinformatics. His research emphasizes practical, open science approaches to bacterial genomics.

Contributing

All Genomicx tools are open source and welcome contributions. If you find a bug, want a new feature, or would like to add support for a new organism or analysis method, open an issue or pull request on the relevant GitHub repository.
