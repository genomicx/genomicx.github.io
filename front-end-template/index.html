Theme Toggle

Dark/light mode with system preference detection. Try the toggle in the nav above.

3-Option Toggle (Dark / System / Light)

class: gx-theme-toggle

Simple Pill Switch

class: gx-theme-switch

JS API

GenomicX.getTheme() // Returns 'dark' or 'light' GenomicX.setTheme('light') // Set explicitly GenomicX.toggleTheme() // Toggle dark <-> light GenomicX.buildThemeToggle() // Returns HTML for the 3-option toggle // Theme is persisted to localStorage key 'gx-theme' // Respects prefers-color-scheme when no manual choice is saved

How It Works

  • Set data-theme="dark" or data-theme="light" on <html>
  • All --gx-* tokens automatically switch values
  • Without data-theme, respects prefers-color-scheme
  • User choice persisted to localStorage as gx-theme
  • Smooth 0.3s transition on all themed elements

Component Patterns

Documented patterns for the three core layout components: nav bar, page title, and footer.

Nav Bar

The nav bar is sticky, uses a blurred backdrop, and includes the GenomicX logo, page links (including Roadmap), a GitHub link, and a theme toggle. On mobile it collapses behind a hamburger menu.

<nav class="gx-nav"> <div class="gx-nav-inner"> <a href="index.html" class="gx-nav-logo"> <svg viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round" style="color: var(--gx-accent)"> <path d="M2 15c6.667-6 13.333 0 20-6"/> <path d="M9 22c1.798-1.998 2.518-3.995 2.807-5.993"/> <path d="M15 2c-1.798 1.998-2.518 3.995-2.807 5.993"/> <path d="M17 6l-2.5 2.5"/><path d="M14 8l-1 1"/> <path d="M7 18l2.5-2.5"/><path d="M3.5 14.5l.5-.5"/> </svg> <span>GenomicX</span> </a> <button class="gx-nav-toggle" aria-label="Toggle navigation" aria-expanded="false"> <!-- hamburger icon --> </button> <ul class="gx-nav-links"> <li><a href="index.html" class="active">Home</a></li> <li><a href="index.html#tools">Tools</a></li> <li><a href="roadmap.html">Roadmap</a></li> <li><a href="about.html">Mission</a></li> <li><a href="https://github.com/genomicx" class="gx-nav-github">GitHub</a></li> <li><button id="theme-toggle" class="theme-toggle-btn">...</button></li> </ul> </div> </nav>

Page Title / Hero

Every page uses a hero or page-title section with a main heading (.accent-text for the highlighted phrase) and an optional subtitle. Use .gx-hero for landing pages or the scoped hero class for sub-pages.

<!-- Landing page hero --> <header class="gx-hero"> <div class="gx-hero-badge"> <!-- icon SVG --> Open source &middot; Privacy-first </div> <h1>Bioinformatics belongs <span class="gx-accent">in the browser.</span></h1> <p>Subtitle text goes here.</p> <div class="gx-hero-actions"> <a href="#" class="gx-btn gx-btn-primary">Primary CTA</a> <a href="#" class="gx-btn gx-btn-secondary">Secondary CTA</a> </div> </header> <!-- Sub-page hero (about, roadmap, etc.) --> <header class="about-hero"> <h1>Page Title <span class="accent-text">highlighted part</span></h1> <p>Brief description of this page.</p> </header>

Footer

The footer sits at the bottom via margin-top: auto and contains the GenomicX tagline plus links to GitHub, the Mission page, and the founder's website.
